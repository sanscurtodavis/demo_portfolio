Stock News Sentiment Analysis

Skills: Sentiment Analysis, Data Integration, Correlation Analysis


For this project, I developed a sentiment analysis tool to evaluate stock news headlines and analyze their correlation with stock price changes. Using Python and the VADER sentiment analysis library, I assessed the sentiment of each headline, classifying them into positive, negative, or neutral categories. I then integrated data from financial APIs to track historical stock prices and used MySQL to store and manage the data. By calculating percentage changes in stock prices and analyzing their correlation with sentiment scores, I provided valuable insights into how news sentiment might impact stock prices. This project not only enhanced my skills in data analysis and web development but also demonstrated the practical application of sentiment analysis in financial markets.
